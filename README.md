# Stylus

Clothing return-risk prediction app built for CS 411 (Spring 2026). Users create a style profile through onboarding, capture clothing items, and receive return-risk predictions.



## Repository

https://github.com/siddvrth/CS411-Stylus.git

## Quick Start

Prerequisites:

- Node.js 18+ and npm

Install dependencies:

```bash
git clone https://github.com/siddvrth/CS411-Stylus.git
cd CS411-Stylus
npm install
```

Run both servers (use two terminals):

```bash
# Terminal 1 — Frontend (port 8080)
npm run dev

# Terminal 2 — Backend (port 3001)
npm run dev:server
```

Then open http://localhost:8080.
