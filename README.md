# Stylus

A React frontend application scaffolded with Vite. This repository contains the UI for the Stylus project (client-side only).

## Repository

https://github.com/siddvrth/CS411-Stylus.git

## Quick start

Prerequisites:

- Node.js 18+ and npm (or use Bun/pnpm if you prefer)

Install and run locally:

```bash
git clone https://github.com/siddvrth/CS411-Stylus.git
cd CS411-Stylus
npm install
npm run dev
```
